ELF Functions Viewer

Прога выводит список глобальных функций которые использует ELF (т.е. из тех что в либе).
Чуть позже функция будет встроена в ElfPack Library Editor.

Запускать так:
ELFFuncV.exe <path_to_elf>

Также можно вручную ассоциировать с файлами .elf
